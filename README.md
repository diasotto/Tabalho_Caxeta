SmartEvent — Aplicativo de Eventos Locais

O SmartEvent é um aplicativo mobile desenvolvido com Ionic + Vue.js 3, criado com o objetivo de conectar pessoas a eventos locais de forma prática, rápida e intuitiva. A proposta do app é centralizar informações sobre eventos como shows, palestras, workshops e atividades esportivas, facilitando a descoberta e o acesso a experiências na região.

A aplicação utiliza uma arquitetura moderna baseada na Composition API, garantindo melhor organização do código, reutilização de componentes e escalabilidade. O sistema conta com múltiplas telas interligadas por navegação fluida, proporcionando uma experiência semelhante à de aplicativos profissionais disponíveis no mercado.

Entre as principais funcionalidades do SmartEvent, destacam-se:

Listagem de eventos com informações detalhadas como data, local e descrição
Tela de detalhes para cada evento, permitindo ao usuário visualizar tudo de forma completa
Sistema de favoritos, onde o usuário pode salvar eventos de interesse
Navegação entre páginas integrada, garantindo usabilidade contínua
Interface moderna e responsiva, adaptada para dispositivos móveis

Além disso, o projeto segue boas práticas de desenvolvimento, incluindo organização por componentes, separação de responsabilidades e uso de versionamento com Git, o que torna o sistema mais profissional e preparado para evolução futura.

O SmartEvent não é apenas um aplicativo funcional, mas também uma demonstração sólida de habilidades em desenvolvimento mobile híbrido, UI/UX e estruturação de projetos modernos, podendo ser facilmente expandido com funcionalidades como login de usuários, integração com APIs reais e geolocalização.
